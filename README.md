## 🛠️ System Overview

This portfolio is built completely from scratch as a lightweight, single-file web application optimized for lightning-fast loading speeds, seamless fluid responsive design, and striking visual storytelling.

### Core Features
* **Futuristic HUD Design:** Built with a localized CSS grid overlay, glowing accent states, and visual telemetry bars tracking core skill brackets.
* **Performance Metrics Terminal:** Features a clean data layout highlighting career stats, remote capabilities, and client growth vectors.
* **Dynamic UX Interactions:** Utilizes a lightweight, performance-tuned `IntersectionObserver` script for fluid scroll-reveal animations without heavy external framework bloat.
* **Fully Responsive Framework:** Built to automatically adjust layouts gracefully between mobile viewports and ultrawide desktop monitors.

## 💻 Tech Stack & Custom Assets

* **Markup:** Semantic HTML5
* **Styling:** Modern CSS3 (featuring CSS Custom Variables, Backdrop-Filters, and Poly-clipping paths)
* **Typography via Google Fonts:** * `Orbitron` (Headings)
    * `Rajdhani` (Body Copy)
    * `Share Tech Mono` (Terminal elements & telemetry data)
* **Interactions:** Vanilla JavaScript (ES6+)
